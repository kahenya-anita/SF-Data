Double response issue: Doubleresponses are fragmented across up to 8 message parts and in some cases the content is identical. As total conversation volume falls, the proportion of doubles is rising. These duplicate responses create noise and a poor user experience, particularl when the message parts are more than 3 from both agent and instant responses.

Intents generating 7–10 part instant responses

These intent categories consistently trigger the longest multi-part responses in the instant response pipeline (116 messages across all months). When doubled, a user may receive up to 16+ message fragments for a single question.

Miscarriage & baby loss Unrelated to pregnancy Baby milestone general Complementary feeding Breastfeeding C-section EDD milestones (twins)
Note on 7–10 part messages: The total of 116 instances per month across these intents indicates the instant response system is not compressing or truncating long-form content. For sensitive topics like miscarriage and baby loss, fragmented multi-part responses may feel disjointed to users who need a clear, consolidated answer.

Instant responses are remarkably consistent: The 1-part / 2-part / 3-part+ split held at 0.7% / 11.5% / 87.8% across all three months without any change.
Agent response complexity is declining over time: The share of 3-part+ agent responses fell from 77.4% in December to 62.7% in February, while 2-part messages grew from 18.7% to 32.5%. This suggests that as fewer agents handle conversations.
